# companion-runtime

Runtime workspace for the Companion project.

## Development

Clone the repository and create changes on a dedicated branch:

```bash
git clone git@github.com:codeinsightlab/companion-runtime.git
cd companion-runtime
git switch -c feature/your-change
```

Project-specific setup, configuration, and run instructions will be added as the runtime is implemented.

## Security

- Keep credentials and local environment files out of Git.
- Use environment variables or a secrets manager for sensitive configuration.
- Review staged changes before every commit.
